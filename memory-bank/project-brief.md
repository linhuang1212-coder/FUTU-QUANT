# FUTU-QUANT 项目定义

## 项目目标
基于富途证券 OpenAPI 的美股全自动量化交易系统，支持日内交易和短线波段。

## 核心约束
- **本金**: $3,000 USD
- **标的**: ETF 为主（杠杆 ETF 优先：TQQQ/SQQQ/SOXL/SOXS/TNA/TZA），精选个股，强信号时可做期权
- **交易频率**: 日内 + 短线波段（不做中长线）
- **自动化**: 全自动交易，无需人工干预
- **运行环境**: 本地 Windows + FutuOpenD 网关
- **风控**: 单笔最大亏损 3-5%，日亏损上限 8%
- **PDT 规则**: 5 天内最多 3 次日内交易（账户 < $25,000）
- **通知**: Telegram Bot

## 策略路线图
1. **第一阶段**: 技术指标 + 量价分析（动量/均值回归/突破）
2. **第二阶段**: 多因子模型增强
3. **第三阶段**: AI/ML 预测模型

## 设计原则
- 模块化架构，策略可插拔
- YAML 配置驱动，不改代码即可调参
- 模拟盘优先验证，一行配置切实盘
- 完整日志记录
